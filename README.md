Plataforma desenvolvida com o objetivo de ser um portal de controle administrativo de assinaturas de clientes. Desenvolvida com as seguintes tecnologias:

Back-end:
* API Rest .net core C#
* Entity Framework
* UnitOfWork
* Repository

Front-end
* Angular 13
* Bootstrap
* Typescript 4.4
* Html

Aplicativo Android
* Arquitetura MVVM
* Retrofit
* LiveData
* Coroutines
* Material Design

Aplicativo IOS
* Ionic Framework 7
* Angular 15
